## 💡 COMMON ACTIONS

### Create New Post

1. Login to admin panel
2. Click "Create New Post"
3. Fill the form
4. Click "Create Post"

### Edit Post

1. Go to "Manage Posts"
2. Click "Edit" on any post
3. Update and save

### View Public Blog

1. Open: http://localhost:5174/blogs
2. Click any blog post
3. Like it ❤️
